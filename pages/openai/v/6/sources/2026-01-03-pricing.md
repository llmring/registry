- gpt-5-nano: Fastest, most cost-efficient version of GPT-5
- gpt-5.2-pro: Version of GPT-5.2 that produces smarter and more precise responses

- gpt-4.1-mini: Smaller, faster version of GPT-4.1
- gpt-4.1-nano: Fastest, most cost-efficient version of GPT-4.1

**Reasoning models:**
- o3: Reasoning model for complex tasks, succeeded by GPT-5
- o3-pro: Version of o3 with more compute for better responses
- o3-mini: A small model alternative to o3
- o4-mini: Fast, cost-efficient reasoning model, succeeded by GPT-5 mini
- o1: Previous full o-series reasoning model
- o1-pro: Version of o1 with more compute for better responses
- o1-mini: Deprecated - A small model alternative to o1

**Specialized models:**
- o3-deep-research: Our most powerful deep research model
- o4-mini-deep-research: Faster, more affordable deep research model
- computer-use-preview: Specialized model for computer use tool

**Codex models:**
- gpt-5.1-codex: A version of GPT-5.1 optimized for agentic coding in Codex
- gpt-5.1-codex-max: Our most intelligent coding model optimized for long-horizon, agentic coding tasks
- gpt-5.1-codex-mini: Smaller, more cost-effective, less-capable version of GPT-5.1-Codex
- gpt-5-codex: A version of GPT-5 optimized for agentic coding in Codex
- codex-mini-latest: Deprecated - Fast reasoning model optimized for the Codex CLI

**Legacy models:**
- gpt-4o: Fast, intelligent, flexible GPT model
- gpt-4o-mini: Fast, affordable small model for focused tasks
