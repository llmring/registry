## Pricing for Tools

Tools are priced at their own rates, applied to the model using them.

| | Free Tier | Paid Tier |
|---|---|---|
| Google Search | 500 RPD free (limit shared for Flash and Flash-Lite). Not available for Pro. | 1,500 RPD free, then $35 / 1,000 grounded prompts. Gemini 3 specific: $14 / 1,000 search queries |
| Google Maps | 500 RPD. Not available for Pro. | 1,500 RPD free (Flash/Flash-Lite), 10,000 RPD free (Pro), then $25 / 1,000 grounded prompts |
| Code execution | Free of charge | Free of charge |
| URL context | Free of charge | Charged as input tokens per model pricing |
| Computer use | Not available | See Gemini 2.5 Computer Use Preview pricing table |
| File search | Free of charge | Charged for embeddings at $0.15 / 1M tokens. Retrieved document tokens charged as regular tokens |

## Pricing for Agents

Agent usage costs are calculated based on the underlying token consumption and usage of the tools.

| Model | Tools |
|---|---|
| Gemini Deep Research Agent | All model inference is charged at standard Gemini 3 Pro list rates, including input, output, and intermediate input / reasoning tokens generated during agentic loops. | Tool usage fees apply per existing pricing structure |

Notes:
- RPD = Requests per day
- Batch API provides 50% discount
- Prices may differ from Vertex AI pricing
